API

I used the USGS Earthquake API to fetch earthquake data.

Visualizations

My site provides an interactive map visualization that shows users the locations of the highest magnitude earthquakes for the selected time period.

JS Libraries

I used Leaflet.JS to create the map visualization. I utilized the Fetch API to retrieve data from the API.

CSS Frameworks

My styles were written using standard CSS syntax without any specific CSS frameworks.

Project Description

With this project, I aimed to solve the problem of Data Overload and Inaccessibility. The API contained a large amount of data that was difficult to understand or access. My site allows users to select specific dates and presents them with earthquake data for that period along with an interactive map showing the strongest earthquakes. I hope that my site simplifies the data access process and prevents users from being overwhelmed with excessive information.
